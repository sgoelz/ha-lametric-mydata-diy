## v0.2.0

First public release of the Home Assistant custom integration for LaMetric My Data DIY feeds.

### Added

- Home Assistant config flow and options flow
- Support for multiple feeds via multiple config entries
- Up to 8 configurable frames per feed
- Per-frame entity, icon, duration, format, prefix and suffix
- Automatic refresh on entity state changes
- Manual refresh service: `lametric_mydata_diy.refresh`
- HACS validation workflow and brand assets

### Improved

- Two-step setup flow with cleaner separation between general settings and frame details
- Better GitHub and HACS repository readiness
- Manifest metadata and validation fixes

### Notes

- Designed for LaMetric `My Data DIY` feeds written to `/config/www/...`
- Useful for common Home Assistant dashboards such as PV power, battery SoC and EV battery state
