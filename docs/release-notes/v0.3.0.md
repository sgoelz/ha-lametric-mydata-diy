# v0.3.0

This release focuses on better day-to-day usability in Home Assistant and on the LaMetric itself.

## Added

- frame presets for common use cases such as power, battery percentage, energy, temperature, voltage, current, and clock
- additional value formats for `temperature`, `voltage`, and `current`
- per-frame hide rules for `0`, empty, and unavailable values

## Improved

- current-time frames now use the standard clock icon `7645`
- updated configuration labels and translations in German and English
- refreshed brand assets
- README updated to match the current config flow and runtime behavior

## Fixed

- hardened startup listener cleanup to avoid Home Assistant restart noise from duplicate one-time listener removal
