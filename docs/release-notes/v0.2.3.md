- automatic minute-based refresh for time frames
- built-in clock icon fallback when time frames use icon `0`

- README updated to cover the current feature set and likely next improvements
- release documentation now reflects the cleaner two-step frame configuration flow

## Fixed

- time frames can now work without a selected entity
- frame logic now treats the current-time preset consistently during rendering and refresh handling
