# v0.2.2

## Added

- new `energy` value format with automatic scaling between `Wh`, `kWh`, `MWh`, and `GWh`

## Improved

- frame configuration is now grouped into collapsible sections in the UI
- frame field labels are now human-friendly in German and English
- German translations now use proper umlauts
- wording around output paths is more portable across Home Assistant installation types

## Fixed

- output path validation is constrained to Home Assistant's `www/...` folder
- feed writes now use unique temporary files to avoid collisions during atomic writes
